HTML — Structure

Element = <start-tag> content </end-tag>. Elements can be nested.

Page Skeleton 
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>My First Page</title>
  </head>
  <body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
  </body>
</html>

CSS — Style

Use inline, internal, or external styles. Prefer external .css for real projects.

<h1 style="color:blue">A Blue Heading</h1>
<p style="color:red">A red paragraph.</p>
<style>
  body{ background: powderblue }
  h1{ color: blue }
  p{ color: red }
</style>
<link rel="stylesheet" href="styles.css">
/* styles.css */
body{ background: powderblue }
h1{ color: blue }
p{ color: red }
Box Model 
p{
  border: 2px solid powderblue; /* line around the element */
  padding: 30px;                /* space between border and content */
  margin: 50px;                 /* space outside the border */
}
